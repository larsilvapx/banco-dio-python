# 🏦 Banco Digital em Python (POO)

Projeto desenvolvido com foco em **Programação Orientada a Objetos (POO)**, simulando um sistema bancário com operações básicas como criação de clientes, contas, depósitos, saques e extrato.

---

##  Objetivo

Este projeto foi criado com o objetivo de praticar e consolidar conceitos fundamentais de desenvolvimento de software, especialmente:

* Programação Orientada a Objetos (POO)
* Organização de código em módulos
* Separação de responsabilidades
* Simulação de regras de negócio reais

---

##  Conceitos aplicados

✔ **Encapsulamento**
✔ **Herança**
✔ **Polimorfismo**
✔ **Abstração**
✔ **Composição entre classes**

---

##  Estrutura do projeto

```
banco-dio-python/
│
├── main.py
│
├── model/
│   ├── Cliente.py
│   ├── Conta.py
│   ├── ContaCorrente.py
│   ├── Historico.py
│   └── Transacao.py
│
└── util/
    └── Menu.py
```

---

##  Funcionalidades

*  Criar cliente (Pessoa Física)
*  Criar conta corrente
*  Depositar valores
*  Sacar valores (com limite e regras)
*  Exibir extrato
*  Listar contas cadastradas

---

##  Como executar o projeto

### 1. Clone o repositório

```
git clone [https://github.com/larsilvapx/banco-dio-python.git]
```

### 2. Acesse a pasta

```
cd banco-dio-python
```

### 3. Execute o projeto

```
python main.py
```
